### React playpen

Learning about:

- React
- NPM  (use `npm run watch` to start watchify)
- ES6  (using 6to5)


Setup using:
    npm install
    npm run watch
    npm test


Installed using:
    
     npm install react browserify watchify 6to5ify  --save-dev -g
     
For babel

     npm install --global --save-dev babelify
